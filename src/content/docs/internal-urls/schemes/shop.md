# `shop` ショップ
主に、スタンプストアなどで使われているようです。
## `shop/sticker/detail/{package_id: string}`
スタンプを開きます。
### Options:
#### package_id: string
スタンプのパッケージIDを指定します。
## `shop/category/{category_id: string}`
指定したカテゴリの人気スタンプランキングを表示します。
### Options:
#### category_id: string
カテゴリーIDを指定します。
## `shop/sticker/author/{author_id: string}`
指定した作者のスタンプ一覧を表示します。
### Options: 
#### author_id: string
作者IDを指定します。
